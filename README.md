#  Workout API Backend

##  Project Description
This is a Flask REST API for a workout tracking system. It allows users to create workouts and exercises, and link exercises to workouts with sets and reps. The project demonstrates the use of Flask, SQLAlchemy, Marshmallow, and Flask-Migrate to build a fully functional backend system.

---

## ⚙️ Technologies Used
- Python 3.8+
- Flask
- Flask-SQLAlchemy
- Flask-Migrate
- Marshmallow
- SQLite
- Pipenv
